# Features

This list describes product modules. For current implementation status, see `docs/STATE.md`.

## Task management

- Create tasks and view task lists
- Inspect task details, including status, input, and output
- Track task status progression (pending -> running -> done)

## Prompt-based creation entry

- Prompt input with mode toggle (general/pro)
- Inspiration prompt examples
- Model selection UI

## Materials workspace

- Conversation-style history of requests
- Asset package views (summary, style, roles, scenes, storyboard)
- Candidate selection and modification input

## Editing workspace

- Shot-based editing view
- Per-shot media preview and controls
- Generate and export actions

## Asset library

- Browse roles, styles, scenes, voices, and templates
- Search, filter, sort, and batch actions

## Space management

- Personal/team space switcher
- Lists of in-progress and completed journeys

## Authentication UI

- Login and registration screens
